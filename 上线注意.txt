1、Guzzle环境要求：

PHP 5.5.0
使用PHP的流， allow_url_fopen 必须在php.ini中启用。
要使用cURL，你必须已经有版本cURL >= 7.19.4，并且编译了OpenSSL 与 zlib。
php扩展必须打开php_openssl
=================================================
2、下载一个cacert.pem，放到PHP安装的根目录下，最好跟php.ini同一个目录，在php.ini文件中配置下路径
[curl]
; A default value for the CURLOPT_CAINFO option. This is required to be an
; absolute path.
;curl.cainfo =
curl.cainfo = "D:\phpStudy\php\php-5.6.27-nts\cacert.pem"
